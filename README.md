# WDINVIR
The simplest 2D simulation of wind currents
For compile and run project you must use command
```
cargo run --release
```
## Controls
`W | ArrowUp` - move up

`A | ArrowLeft` - move left

`S | ArrowDown` - move down

`D | ArrowRight` - move right

`Z` - zoom map

`X` - redo zoom

`Q | E` - rotate map

`V` - change vision
